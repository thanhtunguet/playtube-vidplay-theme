Preparation

First of all, thanks for purchasing this theme. I hope you will love the design of theme. Now it’s turn to see VidPlay live in action.

Download theme
  • Login to your CodeCanyon account.
  • Go to your downloads page.
  • Beside our theme, you can see the download button, click on it, then click on All files & documentation. As shown below:



  • After that, VidPlay theme will be downloaded. Now its turn to install it.

Installation

The process of activating the theme is very simple. In some case, you don’t know about theme installation, you can follow the simple steps below.

  • Now extract the zip file which you have downloaded in the previous process, with the help of ‘WinRar’ or ‘7zip’. The zip file contains the following files and folders



  • Main theme is in Skin folder.
    Open it, there you will see "Themes" folder.
    Now open Themes folder.

  • Now Copy the VidPlay folder and upload it to your website by FTP or with the help of cPanel in the themes directory of your website.


Activating Theme

It’s time to activate the theme via admin panel. To do so, visit the admin panel of your site and choose Design > Theme option from the left sidebar. Now click on Active to activate the theme.




That's it. Now VidPlay theme is activated on your site.
